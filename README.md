# Driver Fatigue Log

WA Commercial Vehicle Fatigue Management app.

## Run the app

Use the **Next.js** app in `app-next/`:

- **Windows:** Double-click **Start Next App.cmd** (or **Start App (Next).bat**) in this folder. It will install dependencies, set up the database, and open http://localhost:3000.
- **Manual:** See **app-next/README.md** for `npm install`, `.env`, Prisma, and `npm run dev`.

## Back up to GitHub

See **HOW_TO_GET_APP_ON_GITHUB.md** for Git setup. To save your latest changes, run **Back up to GitHub.bat** (from File Explorer) or use the commands in that doc.
